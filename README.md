# Tools Workshop

This repository contains the necessary files for the two workshop sessions for the IMAT degree at Universidad Pontificia Comillas.

## Workshop Overview

These workshops are designed to introduce students to essential development tools and workflows that will enhance their productivity and efficiency throughout their studies and professional careers.

## Session 1: Git and Debugging with VS Code

The first session focuses on version control and debugging techniques:

- **Git fundamentals**: Learn the basics of version control, including committing changes, branching, merging, and collaborating with others
- **Debugging with VS Code**: Master the debugging tools in Visual Studio Code, including breakpoints, watch expressions, call stacks, and step-by-step execution

## Session 2: VS Code Extensions, UV, and WSL

The second session covers advanced development environment setup and tools:

- **Useful VS Code Extensions**: Explore and configure essential extensions that will improve your coding experience and productivity
- **Managing Environments with UV**: Learn how to use UV for Python environment and package management
- **WSL (Windows Subsystem for Linux)**: Understand how to set up and use WSL for a Linux development environment on Windows

## Getting Started

The files in this repository are organized to support both workshop sessions. Follow along during the workshops to make the most of these materials.
